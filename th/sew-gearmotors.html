ในฐานะพันธมิตรที่เชื่อถือได้ในอุตสาหกรรมระบบส่งกำลัง WeDrive มีความเชี่ยวชาญในการจัดหาชิ้นส่วนอุตสาหกรรมคุณภาพสูงที่ปรับแต่งได้ตามความต้องการสำหรับการค้าทั่วโลก เราตระหนักดีถึงบทบาทสำคัญของมอเตอร์และเกียร์ทดที่มีความน่าเชื่อถือในสายการผลิตของคุณ เราจึงทุ่มเทเพื่อส่งมอบโซลูชัน SEW-EURODRIVE ของแท้ ซึ่งเป็นที่ยอมรับในด้านความทนทานและความยืดหยุ่นในระบบแบบแยกส่วน

เกียร์ทด - การกำหนดรหัสรุ่น (Type Designation)

ตัวอย่างด้านล่างแสดงโครงสร้างของการกำหนดรหัสรุ่น:

ตัวอย่าง: X3KS250e /HU /B
X ซีรีส์เกียร์อุตสาหกรรม
3 จำนวนสเตจการลดความเร็ว:
  • 2 = 2 สเตจ
  • 3 = 3 สเตจ
  • 4 = 4 สเตจ
K ประเภทเฟืองเกียร์:
  • F = เกียร์เฮลิคอล (เพลาขนาน)
  • K = เกียร์ดอกจอก-เฮลิคอล (เพลาตั้งฉาก)
  • T = เกียร์ดอกจอก-เฮลิคอล (เพลาตั้งฉาก)
S ประเภทเพลาขาออก:
  • S = เพลาตันพร้อมลิ่ม
  • R = เพลาตันแบบเรียบ
  • L = เพลาตันแบบร่องฟัน (Splined)
  • A = เพลาแบบกลวงพร้อมร่องลิ่ม
  • H = เพลาแบบกลวงพร้อมดิสก์หด
  • V = เพลาแบบกลวงร่องฟัน
  • T = เพลาแบบกลวงพร้อมระบบยึด TorqLOC®
  • C = เพลาตันเสริมความแข็งแรงพร้อมลิ่ม
การใช้งานเฉพาะ:
  • B = เกียร์สำหรับลิฟต์แบบถัง (Bucket elevator)
  • C = เกียร์สำหรับระบบยกของ (รอก/กว้าน)
250 ขนาดเกียร์: 100 – 320
e รุ่น X.e (Generation X.e)
HU รูปแบบเสื้อเกียร์ (Housing model):
  • HU = เสื้อเกียร์แบบอเนกประสงค์ (Universal housing)
  • HH = เสื้อเกียร์แนวนอน (Horizontal housing)
  • HA = เสื้อเกียร์สำหรับเครื่องกวน (Agitator housing)
  • HT = เสื้อเกียร์แบบระบายความร้อน (Thermal housing)
  • HC = เสื้อเกียร์สำหรับระบบยก (Hoist housing)
  • HW = เสื้อเกียร์แบบเชื่อม (Welded housing)
B การติดตั้งเกียร์:
  • /B = ติดตั้งแบบขาตั้ง (Foot-mounted)
  • /T = ติดตั้งแบบแขนรับแรงบิด (Torque arm)
  • /F = ติดตั้งแบบหน้าแปลน (Flange-mounted)
