Réducteurs — Clé de désignation (Désignation technique)

Vous trouverez ci-dessous un exemple de décodage de la désignation du modèle :

Exemple : X3KS250e /HU /B
X Série de réducteurs industriels
3 Nombre d'étages de réduction :
  • 2 = deux étages
  • 3 = trois étages
  • 4 = quatre étages
K Type de réducteur :
  • F = Engrenages droits (coaxial)
  • K = Engrenages coniques
  • T = Engrenages coniques
S Exécution de l'arbre de sortie :
  • S = Arbre plein avec clavette
  • R = Arbre plein lisse
  • L = Arbre plein avec cannelures
  • A = Arbre creux avec rainure de clavette
  • H = Arbre creux avec frette de serrage
  • V = Arbre creux avec cannelures
  • T = Arbre creux avec système de montage TorqLOC®
  • C = Arbre plein renforcé avec clavette
- Application :
  • B = Réducteurs pour élévateurs à godets (Bucket Elevator)
  • C = Réducteurs pour applications de levage
250 Taille du réducteur : 100 à 320
e Génération X.e
HU Exécution du carter :
  • HU = Carter universel
  • HH = Carter horizontal
  • HA = Carter pour agitateur
  • HT = Carter thermiquement optimisé
  • HC = Carter pour mécanisme de levage
  • HW = Carter soudé
B Type de montage :
  • /B = Montage sur pattes
  • /T = Avec bras de couple
  • /F = Montage sur bride

Questions fréquentes sur les réducteurs SEW

Q : Quel équipement de SEW-EURODRIVE fournissez-vous ?
R : Nous fournissons une gamme complète de solutions SEW, notamment les réducteurs à engrenages droits, coniques et à vis sans fin des séries R, F, K, S et W, ainsi que les variateurs MOVITRAC® et les moteurs triphasés à haut rendement classe IE3 de la série DRN.

Q : Pouvez-vous aider à décoder les codes techniques (clé de désignation) des modèles SEW ?
R : Oui. Notre équipe technique est spécialisée dans l'identification des codes modulaires (tels que KHF, R..F, RX) et dans la sélection des pièces de rechange appropriées ou des unités d'entraînement prêtes à l'emploi.

Q : Les produits proposés sont-ils originaux ?
R : Absolument. Wedrive Power fournit exclusivement des produits 100 % originaux SEW-EURODRIVE, garantissant la longévité et la fiabilité de vos lignes de production industrielles.

Q : Proposez-vous une expédition internationale pour l'équipement SEW ?
R : Oui. Nous proposons des services d'exportation dans le monde entier, ainsi qu'un emballage professionnel et un support logistique pour les clients industriels du monde entier.
