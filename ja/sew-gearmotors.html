概要と型式

設計とオプション - R、F、K、S、W ギアユニット

動力伝達の信頼できるパートナーとして、WeDrive はグローバル貿易向けに調整された高品質な産業用コンポーネントの提供を専門としています。信頼性の高いギアユニットとモーターが生産ラインにおいて果たす重要な役割を認識し、耐久性とモジュール式の柔軟性で定評のある SEW-EURODRIVE ソリューションの提供に尽力しています。

ギアユニット - 型式指定

以下の例は、型式指定の構造を示しています：

例: X3KS250e /HU /B
X 産業用ギアユニットシリーズ
3 ギアユニット段数:
  • 2 = 2段
  • 3 = 3段
  • 4 = 4段
K ギアユニット設計:
  • F = ヘリカルギアユニット
  • K = ベベルヘリカルギアユニット
  • T = ベベルヘリカルギアユニット
S 出力軸タイプ:
  • S = キー付きソリッド軸
  • R = 滑らかなソリッド軸
  • L = スプラインソリッド軸
  • A = キー溝付き中空軸
  • H = 収縮ディスク付き中空軸
  • V = スプライン中空軸
  • T = TorqLOC® 取り付けシステム付き中空軸
  • C = 強化キー付きソリッド軸
(なし) アプリケーション:
  • B = バケットエレベータ用ギアユニット
  • C = ホイスト（巻上機）用ギアユニット
250 ギアユニットサイズ: 100 – 320 e X.e ジェネレーション HU ハウジング設計:
  • HU = ユニバーサルハウジング
  • HH = 水平ハウジング
  • HA = アジテーターハウジング
  • HT = サーマルハウジング
  • HC = ホイストハウジング
  • HW = 溶接ハウジング
B ギアユニット取付方法:
  • /B = 足取付形 (フット)
  • /T = トルクアーム
  • /F = フランジ

詳細については、ホームページからカタログをダウンロードしてください。

SEW ギアモーター FAQ（よくある質問）

Q: どのような SEW-EURODRIVE 製品を取り扱っていますか？
A: R、F、K、S、W シリーズのヘリカル、ベベル、ウォームギアユニットをはじめ、MOVITRAC® ドライブ、DRN シリーズ IE3 モーターなど、SEW ソリューションの全ラインナップを提供しています。

Q: SEW の型式指定（記号）の解読を手伝ってもらえますか？
A: はい。当社の技術チームは、モジュール設計（例：KHF、R..F、RX など）の特定と、正しいスペアパーツや代替ユニットとの照合を専門としています。

Q: 提供される製品はすべて純正品ですか？
A: もちろんです。Wedrive Power は、お客様の産業生産ラインの耐久性と信頼性を確保するため、100% 純正の SEW-EURODRIVE 製品のみを供給することに注力しています。

Q: SEW 部品の国際物流には対応していますか？
A: はい。世界中の産業クライアント向けに、専門的な梱包と配送サポートを含むグローバル輸出サービスを提供しています。
